視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景的移動速度比前景慢，營造出訪客向下捲動時的 3D 深度與沉浸感。 這種效果源自視覺錯覺：人眼會把近處物體看得更大、也感覺更快，而遠處物體看起來移動較慢。 這種幻覺長久以來被運用在各種媒介的視差技術中，帶來更寫實的效果。 最早的應用可追溯到迪士尼的《白雪公主與七矮人》等傳統動畫，以及《超級瑪利歐》等電玩遊戲。 隨著 CSS 與 HTML 的進步，視差效果逐漸演變成如今常見的網頁設計手法。

2007 年，網頁開發者 Glutnix 在部落格分享了早期的瀏覽器視差滾動範例， 其中包含支援 IE6 等瀏覽器的 JavaScript 與 CSS2 範例與示範。 直到 2011 年 HTML5 與 CSS3 推出後，這項技術才在網頁設計師之間受到歡迎。 支持者認為這是擁抱網路流動性的簡單方式，也運用視差背景來加強互動、提升整體體驗。 不過，普渡大學在 2013 年發表的研究指出：「即便視差滾動強化了某些使用者體驗面向， 它未必能改善整體體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要原因：

  • 有助於講故事

    視差圖像非常適合用來說故事。好的設計就是良好的溝通， 我們在設計網站時，希望與訪客交流，並講述關於產品、服務或品牌的吸引人故事。 視差滾動能讓訪客沈浸在獨特的線上體驗中，甚至能打造單頁網站， 讓使用者不需切換頁面就能讀完完整的品牌故事。

  • 提升使用者參與度

    視差圖像也能提升使用者參與度。降低網站跳出率是網頁設計必須留意的目標之一， 跳出率代表訪客只看一頁就按下返回或關閉分頁的比例。 過高的跳出率會影響搜尋排名，因此值得採取各種方法降低數字。 精心打造的視差頁面能帶來更動態的體驗，讓使用者停留更久。


想在網站加入視差滾動效果時，可以留意以下建議。

  • 選擇性使用

    把視差滾動用在特定元素上效果最佳，不必每個頁面、每個位置都採用。 例如應用在頁首或主標題，或僅放在首頁， 讓視差效果強化網站設計，而不是分散使用者注意力。

  • 壓縮圖像

    視差滾動通常需要多種媒體檔與 CSS 形狀來營造層次感。 若圖片太大導致載入過慢，會犧牲效能與體驗，因此務必壓縮圖像。

  • 運用色彩

    視差滾動不只是前景與背景的圖片，色彩同樣重要。 可以在背景使用較低彩度的色調，並在前景採用鮮明的色彩， 進一步拉出空間深度。
